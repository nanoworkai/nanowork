## Color Scheme

### Light Mode (Current Implementation)

**Backgrounds**:
- Surface 0: `#09090b` (zinc-950)
- Surface 1: `rgba(255,255,255,0.03)`
- Surface 2: `rgba(255,255,255,0.05)`
- Surface 3: `rgba(255,255,255,0.08)`

**Text**:
- Primary: `#ffffff` (white)
- Secondary: `rgba(255,255,255,0.8)`
- Tertiary: `rgba(255,255,255,0.6)`
- Muted: `rgba(255,255,255,0.4)`

**Accents**:
- Primary CTA: `#ffffff` (white background)
- Success: `#22c55e` (green-400)
- Info: `#60a5fa` (blue-400)
- Warning: `#f59e0b` (amber-500)
- Error: `#ef4444` (red-500)

**Borders**:
- Subtle: `rgba(255,255,255,0.1)`
- Medium: `rgba(255,255,255,0.2)`
- Strong: `rgba(255,255,255,0.3)`

## Responsive Behavior

### Desktop (>1024px)
- WelcomeBanner: Full width, 3-column layout
- QuickStart: Full width, all steps visible
- Empty states: Centered, max-width 28rem

### Tablet (768px - 1024px)
- WelcomeBanner: Full width, 3-column layout (may stack)
- QuickStart: Full width, all steps visible
- Empty states: Centered, max-width 24rem

### Mobile (<768px)
- WelcomeBanner: Full width, single-column layout
- QuickStart: Full width, stacked steps
- Empty states: Full width, smaller icons

## Animation & Transitions

### Component Entry
- Fade in over 200ms
- Slight upward movement (10px)

### Dismissal
- Fade out over 150ms
- Slight downward movement (5px)

### Progress Bar
- Width transition over 500ms
- Ease-out timing function

### Button Hover
- Background transition over 150ms
- Scale slightly (1.02x)
- Arrow shifts right 4px

### Collapse/Expand
- Height transition over 200ms
- Chevron rotates 90deg

---

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter/Space to activate buttons
- Escape to dismiss (if focused)

### Screen Readers
- Proper heading hierarchy (h1, h2, h3)
- ARIA labels on icon buttons
- Semantic HTML structure
- Alt text on icons

### Focus Indicators
- Clear focus outline on all interactive elements
- Focus visible on keyboard navigation
- Skip links for main content

### Color Contrast
- All text meets WCAG AA standards (4.5:1)
- Interactive elements clearly distinguished
- Status indicators have icons + text
