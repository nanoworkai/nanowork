## Responsive Breakpoints

### Mobile (<640px)
- 1 column grid
- Compact variant only
- Stack tier badge above ARR
- Hide confidence bars
- Reduce font sizes

### Tablet (640-1024px)
- 2 column grid
- Compact or badge variants
- Show tier badge inline
- Show confidence bars
- Standard font sizes

### Desktop (1024px+)
- 3-4 column grid
- All variants available
- Full feature set
- Hover interactions
- Maximum data density

## Edge Cases

### Very Small ARR (<$1K)
```
[◆ STARTER] $800/YR
```
Show exact amount, no rounding

### Very Large ARR (>$10M)
```
[◉ ENTERPRISE] $5M-10M/YR
```
Use M suffix, round to 1 decimal

### Wide Range (10x difference)
```
[★ SCALE] $50K-500K/YR
          ↑ Indicates high uncertainty
```
Show full range, flag in confidence

### No Confidence Data
```
[▲ GROWTH] $20-50K/YR
(No confidence bar shown)
```
Hide bar, component still works

### Zero Growth
```
↔ Stable revenue          0%
  ↑ Neutral icon
```
Special case for flat revenue

## Component Composition Examples

### Homepage Hero
```tsx
<section>
  <h1>AI-Generated Businesses</h1>
  <p>Browse by revenue potential:</p>
  
  <div className="flex gap-2">
    <ARRDisplay data={starter} variant="badge" />

  </div>
```

### Marketplace Grid
```tsx
<div className="grid grid-cols-3 gap-4">
  {companies.map(company => (
    <CompanyShowcaseCard {...company} />
  ))}
</div>
```

### Detail Page Sidebar
```tsx
<aside>
  <ARRDisplay 
    data={arrData} 
    variant="detailed"
    showBreakdown
    showGrowth
  />

```

### Comparison Table
```tsx
<table>
  <thead>
    <tr>
      <th>Company</th>
```
