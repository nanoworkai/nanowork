# Nanowork Web Project

## No Documentation or Script Files
NEVER create, write, or generate any .md (markdown), .txt (text), or .sh (shell script) files unless explicitly requested by the user. This includes:
- README files
- Documentation files
- Design documents
- Planning documents
- Analysis documents
- Migration guides
- Text files
- Shell scripts
- Any other markdown, text, or script files

This rule applies to all agents, subagents, and workflows working on this project.
